Datos vectoriales de argenmap

No todos los datos que muestra argenmap.jquery en el mapa son del Instituto Geográfico Nacional.
En escalas chicas, Argenmap superpone los datos de calles, rutas y autopistas de OpenStreetMap para darle continuidad a la experiencia del usuario sobre el mapa web aunque el IGN no disponga de datos oficiales a escalas menores a 1:250.000 .

Datos del Instituto Geográfico Nacional

argenmap muestra los datos topográficos del IGN

Los datos vectoriales que argenmap muestra son los de la base de datos Instituto Geográfico Nacional. Estos datos están descompuestos en capas y pueden descargarse libremente desde el sitio del IGN.

Existe una versión antigua de los datos del IGN (denominada SIG250) que también puede descargarse.

Datos de Openstreetmap

argenmap muestra las calles, rutas y autopistas relevadas por los usuarios de OpenStreetMap

Los vectores de calles no son responsabilidad del IGN y los desplegamos sobre el mapa a manera representativa para dar una referencia local en los espacios en los que la escala de captura de los datos del IGN no tienen inferencia.

Openstreetmap es una base de datos mundial que ofrece datos vectoriales de caminos de todo el mundo.

La base de datos es de libre acceso y libre modificación para cada usuario que desee participar aportando sus datos obtenidos con GPS.

Actualmente cuenta con 145.351.000 calles alrededor del mundo y con información (de diferentes precisiones) acerca de más de 221.000 calles dentro del territorio argentino.

Los datos en formato shapefile se pueden obtener desde: http://wiki.openstreetmap.org/wiki/WikiProject_Argentina
