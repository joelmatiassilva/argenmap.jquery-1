agregarMarcador

Esta función te permite agregar un marcador al mapa.

$('#mapa1').argenmap();
$('#mapa1').centro(-34.5722189, -58.44053759999997).zoom(14);
$('#mapa1').agregarMarcador({
	lat:-34.5722189,
	lng:-58.44053759999997,
	contenido:'Hola mundo!'
});

agregarMarcadores

Esta función te permite agregar marcadores a partir de una array de objetos similares a los que recibe .agregarMarcador.

$('#mapa2').argenmap().agregarMarcadores([
	{lat: -25.695278, lng: -54.436667, contenido:'Cataratas del Iguazú!'},
	{lat: -34.05, lng: -59.2, contenido:'Atucha: central eléctrica'},
	{lat: -27.31,  lng: -66.608617, contenido:'Minera Lumbrera: buena mina'}
]);

agregarCapaWMS

Esta función te permite superponer capas WMS sobre las capas base del mapa.

$('#mapa3').argenmap().centro(-25.97885760520047, -59.437950015454135).zoom(7);
$('#mapa3').agregarCapaWMS({
	url:'http://idef.formosa.gob.ar/servicios/wms?',
	capas:'mce_prov_escuelas'

});

agregarCapaBaseWMS

Esta función te permite agregar una capa base al mapa que salga de un servicio WMS.

$('#mapa4').argenmap().agregarCapaBaseWMS({
	url:'http://wms.ign.gob.ar/geoserver/wms?',
	capas:'capabase_msc',
	nombre: 'Capa estilo Manual de Signos Cartográficos'
});
	$('#mapa4').capaBase('Capa estilo Manual de Signos Cartográficos');

agregarCapaKML

Esta función te permite superponer un archivo KML sobre el mapa.

$('#mapa5').argenmap().agregarCapaKML({
	url:'http://sarmientoba.buenosaires.gob.ar/wp-content/themes/plansarmiento/infografia/xml/escuelas.kml'
});
