LIMPIAR OBJETOS DEL MAPA

Esta función permite quitar marcadores, infowindow ... del mapa

limpiar

  • Parámetros específicos:
    • name<string> nombere<string> o array de <string> de Nombre del Storage de los objetos a quitar.
    • first<booleano> Quitar el primer objeto agregado
    • last<booleano> Quitar el último objeto agregado
    • all<booleano> Quitar todos los objetos agregados
    • tag<string> Etiqueta para identificar el objeto
  • Resultado de la función callback: ninguno
Si no se setea ni first, ni last, quita todos los objetos agregados. Podés ver un ejemplo del uso de etiquetas de objetos en el ejemplo de menú contextual.
