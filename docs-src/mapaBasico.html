MAPA BÁSICO para desarrolladores y diseñadores

Incluir jQuery y la API de Google Maps

Primero, tenés que incluir el script de jQuery y el de Google Maps en la sección <head>.

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script type="text/javascript" src="http://maps.google.com/maps/api/js?sensor=false"></script>

Incluir argenmap.jquery

Después incluí argenmap.jquery

<script type="text/javascript" src="http://www.ign.gob.ar/argenmap/argenmap.jquery.min.js"></script>

Ejemplo

argenmap es simple y liviano. Con esta línea incluis el mapa con el mapa del IGN como en este ejemplo.

$('#ejemplo').argenmap();

Como con cualquier plugin de jQuery, tenés que esperar que el DOM esté cargado antes de usar argenmap (el DOM, o el div que vayas a usar).


$(document).ready(function(){
  $("#ejemplo").argenmap();
});

/*
o si preferís

$(function() { 
  $("#ejemplo").argenmap();
});
*/

Importante: argenmap necesita un alto y ancho definidos para el <div>. Así que tenés que definir "height" y "width" del div. Podés hacerlo por css o por style:

Con CSS (como en el ejemplo de esta página)

#ejemplo{
  height: 350px;
  width: 600px;
  margin: 20px auto;
  border: 1px dashed gray;
}

Con el estilo inline

<div id="ejemplo" style="height:350px; width:600px"></div>
