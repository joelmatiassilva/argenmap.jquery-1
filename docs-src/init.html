INICIALIZAR EL MAPA

Esta función permite mostrar un mapa en base a parámetros.

init

  • Parámetros específicos: ninguno
  • Nombre del storage: map
  • resultado de la callback: google.maps.Map

$('#prueba').argenmap({
	accion: 'init',
	opciones: {
		center:[-34, -59],
		zoom:2,
		mapTypeId: 'satellite',
		navigationControl: true,
		scrollwheel: true,
		streetViewControl: true
	}
});
