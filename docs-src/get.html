OBTENER OBJETOS DE MAPA CREADOS POR LA API

Esta función devuelve objetos de Google Maps API creados por las funciones de argenmap (marcadores, rectángulos, etc.)

get

  • Parámetros específicos:
    • name<string> Nombre del Storage del objeto a obtener (el predeterminado es mapa).
    • first<booleano> Devolver el primer objeto agregado
    • all<booleano> Devolver todos los objetos agregados
  • Resultado de la función callback: objeto o array de objetos
Si no se setea ni first, ni all, devuelve el último objeto agregado.
Si name no se setea la función solo devuelve el objeto google.maps.Map instanciado
Podés ver un ejemplo del uso de esta función en la función agregarMarcadores.
