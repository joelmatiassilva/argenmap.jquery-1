AGREGAR MARCADORES AL MAPA

Esta función permite agregar muchos marcadores al map en una sola llamada.
Si la opción clusters está seteada, los marcadores se agrupan en clusters si es necesario. También es posible agregar marcadores a un cluster más adelante.

agregarMarcadores

  • Parámetros específicos:
    • markers: <array de objetos>
      • lat<float> latitud.
      • lng<float> longitud.
      • data<mixed> Data personalizada enviada en los eventos.
      • opciones<objeto> opciones para crear el marcador.
    • cluster: <objeto> Reglas para el manejo de clusters
      • key del objeto: Número del marcador con el cual empezar este cluster (un número entero).
      • content<string> Cadena de texto utilizada para crear el overlay del cluster (<div>...<div>). El pat rón CLUSTER_COUNT es reemplazado por la cantidad de marcadores en el cluster.
      • width<entero> Ancho en píxeles del overlay (obligatorio).
      • height<entero> El alto en píxeles del overlay (obligatorio).
    • radius: <entero> Distancia en píxeles dentro de la cual combinar marcadores en un cluster.
    • maxZoom: <entero> Nivel de zoom máximo hasta el cual utilizar el agrupamiento en cluster.
    • to: <string> Referencia al cluster para agregar nuevos marcadores.

Si clusters está definido:

  • nombre del Storage: cluster
  • resultado de la callback: <string> Referencia al cluster usable para agregar marcadores más tarde.

Si to está definido:

  • Resultado de la callback: <bool> Devuelve true o fale dependiendo de la adición de marcadores al cluster.

Si no está definido:

  • Nombre del storage: marker
  • Resultado de la callback: Array de google.maps.Marker.

Explicación

$('#prueba').argenmap({
	accion: 'agregarMarcadores',
	markers: [
		{lat: -25.695278, lng: -54.436667, data:'Cataratas del Iguazú!'},
		{lat: -34.05, lng: -59.2, data:'Atucha: central eléctrica'},
		{lat: -27.31,  lng: -66.608617, data:'Minera Lumbrera: buena mina'}
	],
	marker: {
		opciones: {
			draggable: false
		},
		events: {
			mouseover: function(marker, event, data) {
				var map = $(this).argenmap('get'),
				infowindow = $(this).argenmap({accion:'get', name:'infowindow'});

				if (infowindow) {
					infowindow.open(map, marker);
					infowindow.setContent(data);
				} else {
					$(this).argenmap({accion:'addinfowindow', anchor:marker, opciones:{content: data}});
				}
			},
			mouseout: function() {
				var infowindow = $(this).argenmap({accion:'get', name:'infowindow'});
				if (infowindow) {
					infowindow.close();
				}
			}
		}
	}
});
