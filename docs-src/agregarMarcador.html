AGREGAR UN MARCADOR AL MAPA

Esta función te permite agregar un marcador al mapa.

agregarMarcador

  • Parámetros específicos:
    • InfoWindow opcional para asociar una InfoWindow al marcador.
  • Nombre del storage: marker
  • resultado de la callback: google.maps.Marker

	$('#prueba').argenmap({
		accion: 'agregarMarcador',
		direccion: "Av. Cabildo 381 Buenos Aires",
		map: {
			center: true,
			zoom: 14
		},
		marker: {
			opciones: {
				draggable: true
			}
		},
		infowindow: {
			opciones: {
				content: 'Hola mundo !'
			}
		}
	});
});
