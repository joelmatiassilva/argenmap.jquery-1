Características de argenmap.jquery

argenmap.jquery es un plugin para jQuery que permite combinar los mapas web del IGN con la API de Google Maps con el fin de obtener beneficio de los datos geoespaciales oficiales del IGN sobre las imágenes satelitales de google.

Multi instancia

Podés usar varios mapas en la misma página.

Encadenable

Podés utilizar argenmap.jquery como cualquier plugin de jQuery. 

$('#prueba').show().argenmap().css('border', '1px solid #000000');

jQuery completo

Con este body html... 

<body>
<div id="prueba1" class="mapa arriba"></div>
<div id="prueba2" class="mapa centro"></div>
<div id="prueba3" class="mapa abajo"></div>
</body>

... este código primero inicializa todos los mapas, y luego agrega un marcador en los dos primeros utilizando 2 selectores diferentes. 

$('.mapa').argenmap({
	accion: 'init',
	opciones:{
		center:[-34, -59],
		zoom:2,
		mapTypeControl: true,
		mapTypeControlOptions: {
			style: google.maps.MapTypeControlStyle.DROPDOWN_MENU
		},
		navigationControl: true,
		scrollwheel: true,
		streetViewControl: true
	}
});
 
$('#prueba2').argenmap({
	accion:'agregarMarcador',
	latLng:[-34.001, -59.03]
});
 
$('.mapa.arriba').argenmap({
	accion:'agregarMarcador',
	latLng:[-34.02, -59.01]
})

Acceso completo a la API de Google Maps

Aunque argenmap simplifica el uso de mapas en web con datos oficiales, también permite el acceso a la API de Google Maps nativamente.
Este ejemplo muestra cómo agregar un marcador utilizando un ícono desde un sprite 

					
$('#prueba').argenmap({
	accion: 'agregarMarcador',
	direccion: "Av. Cabildo 381",
	marker: {
		options:{
			icon: new google.maps.MarkerImage("ui-icons_222222_256x240.png", new google.maps.Size(16, 16), new google.maps.Point((14.5*1), (14.5*10)))
		}
	},
	map: {
		center: true,
		zoom: 14,
		mapTypeId: google.maps.MapTypeId.TERRAIN
	}
});

Contextual

El elemento div que invoca a argenmap es el contexto this de las funciones de callback y de eventos. 

					
$('#prueba').argenmap({
	accion: 'agregarMarcador',
	latLng: [-33, 151],
	map: {
		center: true,
		zoom: 8
	},
	callback: function() {
		$(this).css('border', '1px solid red');
	}
});
